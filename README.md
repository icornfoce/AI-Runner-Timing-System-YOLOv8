 # ระบบจดจำใบหน้า (Face Recognition System)

ระบบนี้ใช้ `OpenCV` และ `face_recognition` เพื่อระบุตัวตนของบุคคลผ่านกล้อง Webcam โดยเปรียบเทียบกับภาพในโฟลเดอร์ `Data/`

## 📁 โครงสร้างโฟลเดอร์
โปรแกรมจะอ่านข้อมูลจากโฟลเดอร์ `Data` โดยแบ่งตามชื่อคนดังนี้:
```
Data/
├── Somchai/
│   ├── image1.jpg
│   └── image2.jpg
├── Somsak/
│   └── photo.png
```
**หมายเหตุ:** โปรแกรมจะใช้ชื่อโฟลเดอร์เป็นชื่อที่แสดงบนหน้าจอ

## 🚀 ขั้นตอนการติดตั้ง

1. **ติดตั้ง Library ที่จำเป็น:**
   เปิด Terminal หรือ Command Prompt แล้วรันคำสั่ง:
   ```bash
   pip install -r requirements.txt
   ```
   *(หากติดตั้ง `face_recognition` ไม่ได้ ให้ตรวจสอบว่ามี Visual Studio C++ Build Tools ติดตั้งอยู่ในเครื่องตัวเลือก "Desktop development with C++")*

## 📸 วิธีใช้งาน

### 1. การเตรียมข้อมูล (Capture Faces)
หากยังไม่มีรูปภาพในโฟลเดอร์ `Data` คุณสามารถใช้สคริปต์ที่เตรียมไว้เพื่อถ่ายภาพหน้าตัวเองได้:
```bash
python capture_faces.py
```
- พิมพ์ชื่อคน
- กด 's' เพื่อบันทึกภาพ (แนะนำให้ถ่ายหลายๆ มุม)
- กด 'q' เมื่อเสร็จสิ้น

### 2. เริ่มการจดจำใบหน้า (Main System)
เมื่อมีข้อมูลในโฟลเดอร์ `Data` แล้ว ให้รันโปรแกรมหลัก:
```bash
python main.py
```
- ระบบจะโหลดใบหน้าจากโฟลเดอร์ทั้งหมด
- หน้าจอจะแสดงกรอบสีเขียวพร้อมชื่อหากพบคนในฐานข้อมูล
- แสดงกรอบสีแดงพร้อมคำว่า "Unknown" หากไม่พบข้อมูล
- กด 'q' เพื่อปิดโปรแกรม

---

## 🛠️ การปรับปรุงประสิทธิภาพ
- หากโปรแกรมกระตุก (Lag) สามารถปรับค่า `fx=0.25` และ `fy=0.25` ในไฟล์ `main.py` ให้เล็กลงได้ (เช่น 0.2) เพื่อลดขนาดภาพที่ประมวลผล
- `tolerance=0.6` ใน `face_recognition.compare_faces` คือค่าความเข้มงวด (ยิ่งน้อยยิ่งเข้มงวด)
